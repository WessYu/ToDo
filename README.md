# Ritmo Habit Planner — ToDo App

Aplicação web responsiva para organização pessoal, hábitos, tarefas, planejamento diário, revisão semanal e diário pessoal.

O projeto foi criado para simular um produto real de produtividade, com interface moderna, dados separados por usuário, persistência local, PWA, backup em JSON e integração opcional com GitHub e IA.

## Visão geral

O **Ritmo Habit Planner** é um app de produtividade feito em React para centralizar hábitos, tarefas, agenda, revisão de progresso e registros pessoais.

A ideia do projeto é ir além de uma lista de tarefas simples: ele organiza o dia do usuário, acompanha frequência de hábitos, permite revisão semanal e mantém os dados separados por conta local.

---

## Funcionalidades

- Cadastro e login local com dados separados por usuário
- Hábitos ilimitados com meta, área, cor e histórico
- Planejamento diário com prioridade, energia, horário e projeto
- Vínculo de tarefas com hábitos para cálculo de frequência
- Painel do dia com progresso, sequência, agenda e intenção diária
- Revisão com métricas, gráfico semanal e leitura rápida
- Diário por data com humor, intenção, vitória e notas
- Backup aberto em JSON com importação e exportação
- Instalação como app via PWA quando o navegador permite
- Integração com GitHub para acompanhar atividade pública
- Integração opcional com IA para sugerir prioridades e planos

## Como rodar localmente

```bash
git clone https://github.com/WessYu/ToDo.git
cd ToDo
npm install
npm run dev
```

Depois acesse no navegador:

```txt
http://localhost:5173
```
