# Ritmo Habit Planner

Aplicação de produtividade desenvolvida com React e TypeScript para organizar hábitos, tarefas, planejamento diário e registros pessoais.

## Funcionalidades

- Cadastro e login local
- Hábitos com meta, área e histórico
- Planejamento diário com prioridade e horário
- Tarefas vinculadas a hábitos
- Painel com progresso e agenda
- Revisão semanal com métricas
- Diário por data
- Importação e exportação de dados em JSON
- Instalação como PWA
- Integração com a API pública do GitHub
- Sugestões opcionais por IA

## Tecnologias

- React
- TypeScript
- Vite
- Lucide React
- LocalStorage
- GitHub API
- OpenAI API

## Executando o projeto

```bash
git clone https://github.com/WessYu/ToDo.git
cd ToDo
npm install
npm run dev
```

A aplicação ficará disponível em `http://localhost:5173`.

## Scripts

```bash
npm run dev
npm run build
npm run preview
```
